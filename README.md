\# 🎓 Student Management System



A full-featured ASP.NET Core MVC application for managing student records efficiently and securely.



\## 🚀 Features

\- Add new students

\- Edit and update student records

\- Delete students

\- Role-based authentication

\- Secure login system

\- Entity Framework Core integration

\- SQL Server database



\## 🛠️ Tech Stack

\- ASP.NET Core MVC

\- C#

\- Entity Framework Core

\- SQL Server

\- Bootstrap



\## 📂 Project Structure

\- Controllers

\- Models

\- Views

\- Data

\- Migrations



\## 🔐 Authentication

Role-based access control implemented.
